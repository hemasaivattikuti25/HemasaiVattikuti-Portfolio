> **AI-Powered Productivity · Zero-Trust Architecture · Multiplayer Collaboration**

A full-stack productivity platform engineered with a **zero-trust backend**. Tasks, habits, Pomodoro sessions, mood logging, and AI-powered journaling — all converging in one unified dashboard.

### 🔑 Core Technical Features

| Feature | Details |
|---|---|
| 🔐 **Zero-Trust Architecture** | Row-Level Security enforced at the PostgreSQL level. Even direct Supabase API calls cannot read another user's data |
| 🧠 **AI Semantic Journal Search** | 768-dimensional Gemini embeddings stored in pgvector. Cosine similarity retrieval finds semantically related entries — understands *meaning*, not just keywords |
| 🔗 **Multiplayer Blend Workspaces** | Collaborative accountability rooms via 6-character join codes. RLS policies handle selective cross-user data access securely |
| ⚡ **Full Productivity Suite** | Task lists, habit streaks, Pomodoro sessions, daily mood logging — one dashboard, zero tool fragmentation |
